# ちいさな竜退治 — レトロRPG（単一HTML）

ドラゴンクエスト風のターン制コマンドバトルRPGです。追加ライブラリ・外部ファイル一切なしの
単一 `index.html` で動きます。ダブルクリックしてブラウザで開くだけで遊べます。

## 遊び方

`index.html` をダブルクリック（`file://` でOK）してブラウザで開くだけ。

### 操作方法

| 操作 | キー |
|------|------|
| タイトル画面からゲーム開始 | Enter / Z / Space |
| 移動（フィールド） | 方向キー ↑↓←→ / W A S D |
| コマンド選択（バトル） | ↑↓ / W S（マウスクリックも可） |
| 決定 | Enter / Z / Space |
| キャンセル（じゅもん→戻る） | X / Esc |
| 効果音のオン・オフ | M |
| ゲームオーバー/クリア後に再開 | Enter / Z |
| ゲームオーバー/クリア後にタイトルへ戻る | X / Esc |

## ゲームの流れ

0. 起動すると**タイトル画面**（夕暮れの空・遠景の城と竜・剣を掲げる勇者のドット絵キービジュアル）が表示されます。**Enter / Z / Space** で冒険開始。
1. **町** からスタート。町に入るとHP/MPが全回復します。
2. **くさはら**（緑タイル）を歩くとランダムに**モンスターとエンカウント**。道（茶色）や町・城の上では出ません。水（青）と山（岩肌のタイル）は通れません。
3. バトルは「**たたかう / じゅもん / にげる**」のコマンド制。
   - じゅもん: メラ・ギラ（攻撃）、ホイミ・ベホイミ（回復）。MPを消費します。
4. 敵を倒すと**経験値とゴールド**を獲得。一定値でレベルアップし、ステータス上昇＋新しい呪文を習得。
5. マップ右下の**城** に到達すると**ボス「りゅうおう」**戦。倒すとエンディング。
6. HPが0になるとゲームオーバー（Enter / Z でリスタート）。

## 実装した主な機能

- タイトル画面（canvasドット絵のキービジュアル＋ロゴ「りゅうおうの ぼうけん」＋点滅する開始プロンプト。Enter/Zで開始）
- 16×12タイルのフィールドマップ探索（草原・道・水・山・町・城をcanvasのドット絵で描画）
- 4方向を向く主人公のドット絵スプライト
- 草原限定のランダムエンカウント（確率16%/歩）
- ターン制コマンドバトル（キーボード＋マウス両対応）
- 呪文システム（攻撃/回復、MP消費）
- HP/MP/こうげき/しゅび/レベル/経験値/ゴールドのステータス成長とレベルアップ
- 雑魚4種（スライム・ドラキー・おばけ・ゴーレム）＋ボス（りゅうおう、簡単なAIあり）
- ボス戦は専用の大きなドット絵グラフィック（通常敵より一回り大きい18×16の描き込み）＋禍々しい専用背景
- ゲームオーバー / クリア処理とリスタート
- WebAudio による外部音源なしの簡易効果音（Mでトグル）
- Canvas 2D によるドット絵モンスター（各4〜6色）＋ドット絵タイルマップ（外部画像ゼロ）

## データの追加方法（改造ポイント）

`index.html` 内の script 冒頭のデータ定義部を編集するだけで拡張できます。

- 敵の追加: `ENEMIES` 配列に1エントリ追加（`MONSTERS` に `rows`(文字列ビットマップ)＋`pal`(文字→色)を足すと専用のドット絵に。無い場合はスライムの絵で代替）。
- タイトル題名の変更: 定数 `TITLE_TEXT` を書き換え（既定は「りゅうおうの ぼうけん」）。
- ボスの見た目: 専用ドット絵 `BOSS_DRAGON`（左半分 `_BOSS_L` を左右反転して生成する18×16）を編集。さらに定数 `BOSS_IMAGE`（PNGのdataURI）を定義すれば、ボス戦でドット絵の代わりに一枚絵を表示する受け皿も用意済み（未定義ならドット絵にフォールバック）。
- 呪文の追加: `SPELLS` に定義し、`LEVELS[n].learn` で習得レベルを指定。
- マップ変更: `MAP` 配列の数字を編集（0草原/1道/2水/3山/4城/5町）。
- 成長曲線: `LEVELS` 配列で各レベルの必要経験値とステータスを調整。

## 既知の制限

- セーブ機能はなし（リロードで最初から）。
- マップは1画面固定（スクロールなし）。町・城はマップ上に1つずつ。
- 敵のAIは簡易的（ボスのみ確率で強攻撃）。武器・防具の購入などショップ要素はなし。
- 効果音はブラウザの自動再生ポリシー上、最初のキー操作以降に鳴り始めます。
