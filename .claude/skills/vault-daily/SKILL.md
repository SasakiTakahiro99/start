# Vault日次作業

このスキルが呼ばれたら、秘書（メインの会話）は以下の手順で各部署に作業を依頼する。

## 1. 常に実行: Vaultの日次整理

`obsidian-scribe` を呼び出し、次を依頼する。
- Vault「SecondBrain」内のタグなし/未整理メモを見つけて整形する
- 重複ノート・孤立ノート（他とリンクされていないノート）を検出し、一覧で報告させる

## 2. 曜日を確認

Bashで現在日時を取得し、今日が月曜日かどうか確認する。

## 3. 月曜日のみ追加実行: 週次リサーチ

今日が月曜日の場合のみ、`researcher` に以下を依頼する。
- AI・LLM・AIエージェント関連の技術動向（直近1週間程度）
- 直近で日本語圏・英語圏で話題になっているトピック（コンテンツ企画・収益化のネタ探し目的。なぜ話題か、どう展開できそうかの視点も含める）

researcherの結果を受け取ったら、`obsidian-scribe` にレポートの保存を依頼する。保存先は `SecondBrain/リサーチレポート/YYYY-MM-DD 週次リサーチ.md`（YYYY-MM-DDは実行日）。

## 4. 報告

各部署から返ってきた結果を秘書が要約し、ユーザーに一つのまとめとして報告する。個々の生の出力をそのまま垂れ流さない。
